### 📈 Analytical Focus: Multiple Regression Analysis
I specialize in constructing predictive models to isolate socioeconomic drivers within complex datasets. In my project on the **World Happiness Report**, I utilized the following model structure:

$$Y_{Happiness} = \beta_0 + \beta_1(GDP) + \beta_2(SocialSupport) + \beta_3(LifeExpectancy) + \epsilon$$

**Advanced Techniques Applied:**
* **Statistical Diagnostics:** Mitigating heteroscedasticity using **HC3 Robust Standard Errors**.
* **Residual Analysis:** Shapiro-Wilk testing for normality and outlier diagnosis to ensure model integrity.
